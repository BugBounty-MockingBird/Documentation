## üîß Local Development

To run the documentation site locally:

```bash
# Install Jekyll (if not already installed)
gem install bundler jekyll

# Navigate to docs directory
cd docs

# Serve the site locally
jekyll serve

# Visit http://localhost:4000
```

## üìù Contributing to Documentation

We welcome contributions to improve our documentation! Please see our [Contributing Guidelines](docs/contributing.md) for details.

To contribute:
1. Fork this repository
2. Create a new branch (`git checkout -b docs/your-improvement`)
3. Make your changes in the `docs/` directory
4. Commit your changes (`git commit -m 'docs: improve getting started guide'`)
5. Push to your branch (`git push origin docs/your-improvement`)
6. Open a Pull Request
